# 图算法的基础

图是一种处理关系型数据的数据结构，主要表示数据间如何关联

举个简单例子

```  
a - b - d - f
  \ | / |
    c   e
```
现在用Python的字典，表示一下这个图关系

```python
graph={
	'a':['b','c'],
	'b':['a','c','d'],
	'c':['a','b','d','e'],
	'd':['b','c','e','f'],
	'e':['c','d'],
	'f':['d']
}
```
这种图被称为，**无向图**，因为它没有箭头指明其方向

那么什么是**有向图**呢？

```
a-> b<->
|     |
|-> c<->
```
这里表示了关系的方向，a 可以关注，b,c，但是b，c不关注a，并且b，c互相关注

用Python的列表表示

```python
graph={
	'a':['b','c'],
	'b':['c'],
	'c':['b']
}
```
