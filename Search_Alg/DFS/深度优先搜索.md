# 深度优先搜索算法详解

## 解析

在二叉树算法中，深度优先算法，其实就是

当找到父节点通向子节点的边时，不继续寻找其他子节点，而是以发现的子节点为起点，继续寻找子节点的子节点，这里可被成为孙节点，如此反复，直到没有可以被发现的节点时，将会回溯上一个父节点，继续寻找没有遍历过的子节点，直到全部遍历完成。


```
          1
        /   \
       2     3
      / \   / \
     4   5 6   7
    / \    
   8   9

遍历出来的结果就是

[1,2,4,8,9,5,3,6,7]
```
